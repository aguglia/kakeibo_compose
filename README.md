# 📊 家計簿アプリ (Kakeibo Compose) - v1.0

Jetpack Compose と Room データベースを採用した、シンプルで堅牢な Android 向けモダン家計簿アプリケーションです。

## 📲 インストール方法（実機での動かし方）

本プロジェクトのリリースビルド（APKファイル）を Android 実機にインストールして使用する手順です。

### 1. APK ファイルのダウンロード
- 本リポジトリの右側メニューにある **[Releases]** セクション、または以下のリンクから最新バージョンのページを開きます。
  - 👉 **[最新のリリース (Releases) ページはこちら](https://github.com/aguglia/kakeibo_compose/releases)**
- ページ下部の「Assets」エリアにある **`.apk`** ファイル（例: `Kakeibo_v1.0.apk`）をタップして、Android 端末に直接ダウンロードします。

### 2. インストール手順
1. ダウンロードした `.apk` ファイルをタップして開きます。
2. 初回インストール時、Android のシステムから「**不明なアプリのインストール**」（提供元不明のアプリの許可）の警告画面が表示される場合があります。
3. 画面の指示に従って設定を開き、ファイルのダウンロードに使用したブラウザアプリ（Chromeなど）やファイル管理アプリに対して**一時的にインストールを許可**（トグルをON）してください。
4. 再びインストール画面に戻り、「インストール」をタップすると、スマホのホーム画面にアプリが追加されます。

> ⚠️ **注意点**
> - 本アプリは自己署名パッケージであるため、Play プロテクトなどのセキュリティ警告が表示されることがありますが、開発用の安全な成果物ですのでそのまま進めて問題ありません。

## 🌟 バージョン 1.0 の主な機能

### 1. 柔軟な実績確認・分析画面（新機能）
- **月間 ↔ 年間モードのシームレス切り替え**
  - 月単位での細かい収支追跡はもちろん、年単位での累計実績（今年1年間で何にいくら使ったか）を1画面で俯瞰できます。
  - 選択した期間に応じて、カレンダーコントロールが自動で「月進退」「年進退」に最適化されます。
- **支出 ↔ 収入一覧のトグル切り替え**
  - 下部のカテゴリ内訳リストは、見たい収支タイプにワンタップで連動します。
- **オールインワン・サマリーカード**
  - 画面を切り替えることなく「実際の総収入」「総予算」「実際の総支出」「予算残高（黒字/赤字）」を常に最上部に常時ホールド表示。家計のトップラインとボトムラインを同時に把握できます。

### 2. 視覚的な進捗インジケーター
- 予算に対する進捗度（プログレスバー）を視覚的に表現。
- **支出モード**: 予算消費量に合わせて3段階（安全：緑 ➔ 注意：オレンジ ➔ 超過：赤）にアラート色が変化。
- **収入モード**: 目標達成に向けたクリアなブルーに固定。

### 3. 堅牢なデータ設計と自動履歴連動 (UPSERT)
- 中カテゴリの「新規追加」と「既存編集」のロジックを一本化（`saveMiddleCategoryWithBudget`）。
- カテゴリを新設したその瞬間から、当月の履歴テーブル（`budget_history_table`）に「最初の歴史（初期予算）」を確実に自動インサート。翌月以降の予算自動引き継ぎクエリ（過去直近の遺産を参照する設計）のスタート地点を100%保証します。
- `is_system` フラグによるフィルタリングを徹底し、収入一覧から「初期費用」などのシステムノイズを完璧に遮断。

## 🛠️ 技術スタック
- **UI**: Jetpack Compose (Material 3)
- **Architecture**: MVVM Architecture + Repository Pattern
- **Asynchronous**: Kotlin Coroutines + Flow (StateFlow / collectAsState)
- **Database**: Room Persistence Library (SQLite)
- **Dependency Injection**: Hilt / @Inject

## 📂 データベース構造
- `kakeibo_table`: 日々の取引明細データ（システムデータ除外用 `is_system` 完備）
- `middle_category_table`: 収支を司る中カテゴリ（支出/収入フラグ `is_income` 管理）
- `sub_category_table`: より詳細な小カテゴリ
- `budget_table`: 各カテゴリの最新の標準予算
- `budget_history_table`: **[コア]** 年月（`yyyy-MM`）ごとの予算の歴史を保持。過去の予算の振り返りや、翌月への予算引き継ぎロジックの基盤。
